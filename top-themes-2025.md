## Theme Selection Guide

### Choose Dark Themes When:
- Working in low-light environments
- Extended coding sessions (reduces eye strain)
- Using OLED/AMOLED displays (battery savings)
- Preference for modern, developer-focused aesthetic

### Choose Light Themes When:
- Working in brightly lit environments
- Giving presentations or screen sharing
- Working with printed documents alongside code
- Matching operating system light mode

### Popular Combinations (DHH's Omakub recommendations):
1. **Tokyo Night** (default) - Modern, balanced
2. **Gruvbox** - Retro warmth
3. **Everforest** - Nature-inspired calm
4. **Rosé Pine** - Aesthetic minimalism
5. **Catppuccin** - Pastel comfort
6. **Nord** - Arctic cool
7. **Kanagawa** - Artistic uniqueness
